GoSSH
=====

###Instruction
    
usage: java -jar GoSSH.jar -f -u [-p] -help  
 -c,--command <arg>    Command  
 -f,--file <arg>       IP Address list file  
 -help                 Print this help  
 -p,--password <arg>   Password  
 -u,--username <arg>   Username  
 
###IP Address File
one ip per line
